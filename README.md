 # ☕ Kae’s Café

A luxurious Paris-inspired café website built with React and modern web technologies.  
Kae’s Café combines elegant UI design and responsive layouts.



### 🌸 Features

- ✨ Elegant Parisian-inspired landing page
- 📱 Fully responsive mobile design
- 🍰 Beautiful café showcase cards
- ☰ Accessible hamburger navigation menu
- 🎨 Modern pink luxury aesthetic



### 🛠️ Built With

- React
- React Router
- CSS3
- Vite
- JavaScript (ES6+)

## 🚀 Installation

Clone the repository:

```bash
https://github.com/KunnikarB/kaes-cafe
```

Go into the project folder:

```bash
cd kaes-cafe
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
